About program

SMDownloader

Phần mềm SMDownloader cho phép bạn tải xuống video từ các trang mạng xã hội: Youtube, Facebook, Tiktok và bạn cũng có thể mở trình phát video trên phần mềm để xem video vừa tải hoặc video bất kì. Ứng dụng được hỗ trợ và cho phép chạy trên hệ điều hành Windows và Linux. Ứng dụng này được phát triển bởi nhóm 31


Download phần mềm

Hướng dẫn cài đặt:

Đối với  Windows

Bước 1: Cài đặt Python
https://www.python.org/downloads/
Bước 2: Cài đặt Pip
https://pip.pypa.io/en/stable/installation/
Bước 3: Cài đặt ffmpeg
https://github.com/BtbN/FFmpeg-Builds/releases/download/latest/ffmpeg-master-latest-win64-gpl.zip
Bước 4: Cài đặt thư viện 

    Tkinter: pip install tk 
    Customtkinter: pip install customtkinter 
    PyQt5: pip install PyQt5 
    yt_dlp: pip install yt_dlp 
    tqdm: pip install tqdm 
    pillow: pip install pillow

Bước 5: Tải về file zip giải nén và chạy file main.exe để chạy chương trình

Tải xuống cho Windows

Đối với  Linux

Bước 1: Cài đặt Python 

    sudo apt install software-properties-common
    sudo add-apt-repository ppa:deadsnakes/ppa
    sudo apt update 
    sudo apt install python3.12
Bước 2: Cài đặt Pip 
sudo apt install python3-pip
Bước 3: Cài đặt ffmpeg
Tải file zip ffmpeg ở đây
Giải nén file zip

Sao chép 3 file từ ffmpeg-master-latest-linux64-gpl/bin và dán vào /usr/bin 

    sudo cp /path/to/ffmpeg /usr/bin
    sudo cp /path/to/ffplay /usr/bin
    sudo cp /path/to/ffprobe /usr/bin
Bước 4: Cài đặt thư viện

Tkinter: sudo apt-get install python3-tk
Customtkinter: python3 -m pip install customtkinter
PyQt5: python3 -m pip install PyQt5
yt_dlp: python3 -m pip install yt_dlp
tqdm: python3 -m pip install tqdm
pillow: sudo apt-get install python3-pil python3-pil.imagetk
Thư viện phát video: sudo apt install gstreamer1.0-libav gstreamer1.0-plugins-good gstreamer1.0-plugins-bad gstreamer1.0-plugins-ugly

Bước 5: Tải về file tar giải nén chương trình và chạy file main để chạy chương trình

Tải xuống cho Linux
